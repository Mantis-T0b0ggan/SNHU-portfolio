In IT-270 Website Design I was asked to create a website of my choosing. I decided to create a website for a friend who was starting his own lobster company. The pages are created using HTML5 and given style using CSS. The website contains multiple pages such as an 'About us' and 'Bait Prices' page. The Bait Prices page has a google spreadsheet attached that allowed the owner of the website to update their bait prices quickly by updating the spreadsheet. The website was created from scratch so all the HTML and CSS coding that was done to create the site was done by me.
